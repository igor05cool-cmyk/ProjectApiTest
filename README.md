# ProjectApiTest
Учебный проект по тестированию апи разных сервисов
Перед запуском не забыть установить pytest
Установить библиотеку requests
Сам запуск тестов производить через команду в терминале  "python -m pytest -s -v" - Запуск всех тестов
Запуск конкретных тестов (пример команды - "pytest -k "create_new_place" -s -v")
